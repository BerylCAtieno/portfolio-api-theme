## Overview

This is a personal portfolio website styled in a RESTful API theme

## Prerequisites

- Node.js (v18 or later recommended)
- npm

## Installation and Set-Up

1. Fork then Clone the Repo

```bash
git clone https://github.com/yourusername/portfolio-api-theme.git
cd portfolio-api-theme

```

2. Install dependencies

```bash
npm install
```

3. Run development server

```bash
npm run dev
```
